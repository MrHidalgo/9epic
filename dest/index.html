Delivered in 4 weeks, it’s your fastest path to value in AWS.... guaranteed!

Assess

2 days

Collaborative and guided Discovery Workshop to assess the current state, understand business objectives, and to identify key drivers and requirements for target state

  • Review current state architecture and infrastructure code
  • Identify early adopters (applications to be migrated to cloud and/or to be built natively in Cloud)
  • Finalize the key foundational cloud capabilities for enterprise-wide use
  • Designate stakeholders for design & build reviews

Design

3 days

Target state Design Workshops to define a flexible and future proof architecture for each of the foundational capabilities aligned with AWS Well-Architected Framework

  • Review and tailor out-of-the-box design blueprints that leverage industry best practices
  • Comprehensive policies and controls aligned to CIS, NIST CSF, PCI-DSS
  • Centralized mechanism to manage secrets, intelligently detect & respond to threats, identify vulnerabilities and secure end-points
  • Centrally managed just-in-time and just-enough role-based access control that seamlessly works across on-prem and public cloud
  • AWS account framework that is aligned to the environment strategy of the applications, fulfills the financial accounting needs of the organization and minimizes the blast-radius
  • Comprehensive hybrid cloud network architecture that is highly resilient, secure and scalable
  • Operational capabilities that are fully automated (including image management, patch management, and private DNS)
  • Consistent enforcement of guardrails, controls, quotas, budgets, etc. across the cloud footprint to ensure compliance with IT policies and regulatory requirements
  • Centralized monitoring & logging, unified visualization, and correlation of different security & operational events
  • Centralized control of data protection from failures, security breaches, data leaks and disasters
  • 3rd party tool integrations

Build

5-10 days

Implement a fully operational and production-ready cloud foundation for enterprise-wide use using infrastructure as code integrating the right AWS services and 3rd party products

  • Gather required configuration information
  • Deploy selected capabilities and features in a “cloud-lab” environment
  • Demo & walk-through of foundational capabilities to key stakeholders

Validate

5 days

Perform testing, validation, and production readiness certification in preparation for application onboarding

  • Validate the deployed cloud foundation is configured appropriately
  • Perform comprehensive end-to-end testing to certify all the capabilities deployed are production ready
  • Provide comprehensive documentation of built out cloud foundation (architecture, policies, and controls)
  • Train cloud engineering and operations teams
  • Sign-off and handover
